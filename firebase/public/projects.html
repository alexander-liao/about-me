Programming Languages

I have made several programming languages, two of which are particularly notable. They are both designed for golfing; that is, writing short programs to perform a given task. They are PostL (post-fix notation based) and Anyfix (any-fix notation based, which is one of my own concepts).

  • PostL (Github)
    • Postfix Notation
    • Has a readable version and a golfed version
    • Intuitive coding structure and simple syntax
  • Anyfix (Github)
    • Anyfix Notation - Allows for all 3 types of notation (postfix, prefix, infix)
    • Stores instructions that require more parameters than are currently present until there are enough parameters
    • Designed for golfing
    • Less intuitive syntax and structure but much more flexible
  • BrainF**k Extensions (Github)
    • BrainF**k 1: The Original BF with 8 commands
    • BrainF**k 2: Adds repetition to avoid needing the same command repeated over and over. Supports comments
    • BrainF**k 3: Adds built-ins to do common tasks like read all input
    • BrainF**k 4: Adds function syntax to define functions
    • Overall: Each program in a BF version can be converted into a valid program in a lower BF version, allowing everything to eventually reduce down to original BF code. This makes coding in BF a lot simpler and more intuitive
  • Billiards Ball Machine (Github)
    • Esolang; designed for fun and not practicality
    • Mostly intuitive command structure with values represented as "billiard balls" moving through a ball machine
  • KeyScript (Github)
    • A scripting language designed for interacting with Keyboard Events
    • Mockup version of something similar to AutoHotKey
    • Easy to use and intuitive syntax

Other Notable Programs

  • Implode an ASCII-art box (StackExchange)
    • Takes an ASCII-art box as input
    • Moves all of the edges in to make a cool ASCII-art output
    • Made for a programming challenge to make the shortest program
  • Name an acyclic hydrocarbon (StackExchange)
    • Takes an ASCII-art representation of an acyclic hydrocarbon
    • Outputs its proper IUPAC name
    • Made for a programming challenge to make the shortest program, but turned out to be very difficult even without considering that
